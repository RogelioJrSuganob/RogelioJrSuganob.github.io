Selected Work

QueueFlow

Built a real-time queue management system that simulates how service counters in banks or offices handle customer flow. The application features an admin panel for controlling queue progression and dynamically managing service windows, alongside customer and display views that update instantly across devices using WebSockets. It includes session-based admin authentication, a proper reset state, and a fully responsive, centered UI. Developed with React, Node.js, Express, and Socket.IO, the project focuses on real-world behavior, multi-client synchronization, and clean user experience design.
